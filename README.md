# Miro

_Prototype._

## Dependencies

- [RethinkDB](http://rethinkdb.com)
- [iojs](http://iojs.org/)

## Develop

```bash
$ git clone git@bitbucket.org/mckaysoftware/miro.git
$ cd miro
$ atom .
$ npm run watch
```

## Deploy

```bash
# Only once, follow instructions
$ npm run deploy auth

# To deploy
$ npm run deploy [b[eta]|p[roduction]|t[est]]
```
